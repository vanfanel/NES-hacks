*KIWI KRAZE TITLE/TEXT SCREEN RESTORATION BY VANFANEL IN 2026*
**************************************************************

What does this do?
==================

It restores the name "Newzealand Story" to both the title screen
and the pre-title copyright text screen.
"Kiwi Kraze" sounds absurd to me and I don't understand why it was
made into that for the USA release of the game, since the original
is a name everybody knowed and loved back then thanks to the
classic arcade this fantastic NES game was based on.

How to use this?
================

Simply use your favorite patcher and apply the .bps patch on top of
the No-Intro ROM which is called:
"Kiwi Kraze - A Bird-Brained Adventure! (USA).nes"
CRC32: ed85cefa
SHA-1: 21c458b9a7192e0ca7b99778ab398cc02cd88a75

How was it done?
================

Since I had never done a (public) NES hack before, I simply brute-forced
my way on the .NES ROM using Hexedit on GNU/Linux, after finding out the
related tile areas and the tilemap hex strings using Mesen's viewers for
exploring memory, tiles and tilemaps.
All in all, this small modification implied bringing tiles from the PAL
release, small palette adjustmenst and of course moving around tile indexes.
It was somewhat fun!
